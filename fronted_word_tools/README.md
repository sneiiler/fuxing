# FuXing Word 插件

这是一个基于NetOffice框架开发的Microsoft Word COM插件，成功复现了Reference__WPS_VSTO项目的核心功能，并采用现代化的AntdUI界面库。

## ✅ 已实现的功能特性

### 🤖 AI文本处理
- **选中文本纠错**：使用AI对选中的文本进行纠错和优化
- **全文文本纠错**：使用AI对整个文档进行文本纠错和优化

### 📋 标准校验
- **引用标准校验**：校验选中文本中引用的标准是否正确
- **智能批注**：自动添加带有"AI助手"标识的校验结果批注

### 📊 表格格式化
- **选定表格格式化**：格式化当前选中的表格样式
- **全文表格格式化**：格式化文档中的所有表格样式
- **智能样式**：自动设置边框、字体、行高、标题行样式等

### ⚙️ 配置管理
- **配置文件**：在文档目录自动保存配置文件(office_tools_config.json)
- **服务器配置**：支持LLM服务器和标准校验服务器的IP/端口配置
- **现代化界面**：使用AntdUI提供美观的配置界面

### 🎛️ 用户界面
- **Ribbon界面**：在Word中添加自定义"福星"选项卡
- **右键菜单**：在文本和表格右键菜单中添加快捷操作
- **现代化设计**：使用AntdUI提供的现代化组件
- **任务面板**：可停靠的工具面板，方便快速操作

### 🌐 网络功能
- **HTTP请求**：支持与LLM服务器和标准校验服务器的网络通信
- **JSON处理**：使用Newtonsoft.Json进行数据序列化/反序列化
- **错误处理**：完善的网络请求错误处理和超时控制

## 📁 项目结构

```
FuXing/
├── FuXing.cs                       # 主插件类 - NetOffice COM插件
├── ConfigLoader.cs                 # 配置加载器 - JSON配置管理
├── NetWorkHelper.cs                # 网络请求辅助类 - HTTP通信
├── SettingFormAntd.cs              # AntdUI设置窗体
├── AboutFormAntd.cs                # AntdUI关于窗体
├── AIToolUserControlAntd.cs        # AntdUI工具面板
├── Properties/
│   └── AssemblyInfo.cs             # 程序集信息 (已配置COM)
├── Resources/                      # 图标资源文件夹
├── PostBuild.bat                   # 自动注册脚本
├── RegisterPlugin.bat              # 手动注册脚本
├── UnregisterPlugin.bat            # 卸载脚本
└── README.md                       # 说明文档
```

## 🔧 配置文件示例

插件会在`我的文档`目录下自动创建`office_tools_config.json`配置文件：

```json
{
  "llmServerIP": "127.0.0.1",
  "llmServerPort": 11434,
  "UpdatePort": 11450,
  "OtherPort": 0,
  "CheckStandardIP": "192.168.1.1",
  "CheckStandardPort": 80
}
```

## 📝 使用说明

1. **安装插件**：配置Post-Build自动注册
2. **重启Word**：插件注册后需要重启Word
3. **查看界面**：
   - Word Ribbon中的"福星"选项卡
   - 文本/表格右键菜单中的快捷操作
   - 可停靠的工具面板
4. **配置服务器**：点击"配置"按钮设置服务器地址

## ⚠️ 注意事项

1. 需要安装Microsoft Word
2. 需要**管理员权限**来注册COM组件
3. 插件注册后需要重启Word才能生效
4. 确保.NET Framework 4.7已安装
5. **推荐以管理员身份运行Visual Studio**以启用自动注册
